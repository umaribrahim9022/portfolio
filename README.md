# Umar Ibrahim — Portfolio

## Files
- `App.jsx` — React portfolio (main component)
- `main.jsx` — React entry point
- `index.html` — HTML version (standalone, no setup needed)
- `Umar_Ibrahim_CV.pdf` — CV file

## React Setup (recommended)
```bash
npm install
npm run dev
```
Then open http://localhost:5173

## HTML Version (quick open)
Just open `index.html` in any browser — no setup needed.
